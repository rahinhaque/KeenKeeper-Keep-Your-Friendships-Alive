# 🌟 KeenKeeper

**Keep Your Friendships Alive!**

## 📝 Description

KeenKeeper is a beautiful, modern application designed to help you nurture and maintain your relationships with thoughtful reminders and tools. Track your interactions with friends, set goals, and keep your friendships thriving!

## 🛠️ Technologies Used

- **Next.js 16** - React framework for production
- **React 19** - UI library
- **Tailwind CSS** - Utility-first CSS framework
- **Recharts** - Charting library for data visualization
- **React Spinners** - Loading animations
- **localStorage** - For data persistence

## ✨ Key Features

1. **Friend Management** - View and manage your friends list with detailed profiles, including status indicators (Overdue, Almost Due, On-Track) and tags
2. **Interaction Timeline** - Log and track interactions (Call, Text, Video) with friends, filter by type, and view your complete history
3. **Friendship Analytics** - Visual analytics showing breakdown of interaction types using beautiful pie charts

## 🚀 Getting Started

First, run the development server:

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result!

## 📱 Features Overview

- **Home Page**: Friends list grid with loading indicator (HashLoader)
- **Friend Details**: Complete friend profile, quick check-in buttons (Call/Text/Video), and action buttons
- **Timeline**: Complete interaction history with filter functionality
- **Stats**: Analytics dashboard showing interaction breakdown
